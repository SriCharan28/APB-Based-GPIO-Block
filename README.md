# APB-Based-GPIO-Block
This repository contains the implementation of APB Based GPIO Block in Verilog using Xilinx Vivado.

The APB_GPIO Folder contains the following :-
1) "APB_GPIO.v" - this file contains the top design code of APB Based GPIO Block, where underlying components like IO Interface, Auxilary Interface, APB Slave Interface and GPIO Register are integrated.
2) "APB_GPIO_TB.v" - this file contains the testbench code of APB Based GPIO Block.
   
The APB_Slave_IF Folder contains the following :-
1) "APB_Slave_IF.v" - this file contains the top design code of APB Slave Interface.
2) "APB_Slave_IF_TB.v" - this file contains the testbench code of APB Slave Interface.
  
The Auxilary_IF Folder contains the following :-
1) "Auxilary_IF.v" - this file contains the top design code of Auxilary Interface.
2) "Auxilary_IF_TB.v" - this file contains the testbench code of Auxilary Interface.

The IO_IF Folder contains the following :-
1) "IO_IF.v" - this file contains the top design code of Input-Output Interface.
2) "IO_IF_TB.v" - this file contains the testbench code of Input-Output Interface.

The GPIO_Register Folder contains the following :-
1) "GPIO_Register.v" - this file contains the top design code of GPIO Register.
2) "GPIO_Register_TB.v" - this file contains the testbench code of GPIO Register.
